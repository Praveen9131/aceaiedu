Course Plan

Weeks 1-2: Introduction to Machine Learning & Statistics

  • What is Machine Learning? Types: Supervised, Unsupervised, Reinforcement
  • Basic statistics: mean, median, mode, variance, standard deviation
  • Project: Data exploration and summary statistics in Python

Weeks 3-4: Data Preprocessing & Exploratory Data Analysis

  • Data cleaning, handling missing values, encoding categorical data
  • Data visualization with Matplotlib and Seaborn
  • Project: Visualize and preprocess a real-world dataset

Weeks 5-6: Supervised Learning - Regression & Classification

  • Linear and Logistic Regression
  • Decision Trees, k-NN, SVM basics
  • Project: Predict housing prices and classify iris species

Weeks 7-8: Model Evaluation & Feature Engineering

  • Train/test split, cross-validation, metrics (accuracy, precision, recall, F1)
  • Feature scaling, selection, and engineering
  • Project: Build and evaluate a classification pipeline

Weeks 9-10: Unsupervised Learning & Advanced Topics

  • Clustering: k-Means, Hierarchical, DBSCAN
  • Dimensionality reduction: PCA, t-SNE
  • Project: Segment customers using clustering

Weeks 11-12: Capstone Project & Model Deployment

  • End-to-end ML project: from data to deployment
  • Model deployment basics (Flask, Streamlit, or cloud options)
  • Project: Deploy a trained model as a web service
